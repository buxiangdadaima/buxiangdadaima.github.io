查找端口是否被使用

netstat -lnp | grep port

[root@izuf6ifqt8vh2gzameq5i5z ~]# netstat -lnp | grep 3306
tcp        0      0 0.0.0.0:3306            0.0.0.0:*               LISTEN      1808/mysqld

其中 1808 为 pid

结束进程

kill -option pid

一般结束进程 option 不宜使用 -9,因为其不会触发 shutdown hook，导致程序非正常关闭，需根据情况使用。
