Vim 学习笔记

2019.07.18

作为一只程序猿，怎么能不懂一点 Vim。 学习 Vim，可在终端下输入 vimtutor 即可看到彩蛋。

移动光标

  • h 左移
  • j 下移
  • k 上移
  • l 右移

hjkl - 左下上右

编辑操作

  • i 插入（insert）
  • a 行后添加（append）

删除操作

光标

  • dw 删除光标处一个单词 d-delete w-word
  • d$ 删除光标处到行末 $-通配符行末

d number motion 可在 d 后添加数字，表明重复次数，

行

  • dd 删除整行
  • 2dd 删除两行

撤销操作

  • u 撤销上一次命令
  • U 撤销一行上的所有操作

替换字符操作

  • r + 替换字符 替换对应光标处字符，r-replace

粘贴操作

  • p 粘贴上一次删除内容 p-paste

显示光标当前文件所在位置以及文件状态

  • CTRL + G

移动操作

  • G 移动到文件底部
  • gg 移动到文件头部

搜索操作

  • / + 内容

n-下一个 N-上一个

替换内容操作

  • :s/old/new/g

old 原先内容 new 新内容 g-global 全局替换（不加则替换第一个） c-confirmation（确认提示）

复制操作

  • y

设置行号显示

需在命令模式下输入

  • set number

退出操作

需在命令模式下输入

  • q 退出 q-quit
  • w 保存
  • ! 不保存
